(LAMP, skype, teamviewer, google-chrome)


Give the execution right to it.
------------------------------
```
chmod +x install.sh
```
Then run it as sudo user.
-------------------------
```
sudo ./install.sh
```


How to uninstall:
-------------

Give the execution right to it.
------------------------------
```
chmod +x uninstall.sh
```
Then run it as sudo user.
-------------------------
```
sudo ./uninstall.sh
```
